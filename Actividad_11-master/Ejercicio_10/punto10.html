TARGET

Siempre se aconseja que para mejorar la accesibilidad de las páginas web se deben emplear los estándares. Respecto al HTML, en la actualidad se debe emplear XHTML 1.0 Strict, pero esta elección puede implicar algunos problemas a los desarrolladores, ya que descubrirán que algunos elementos y atributos de HTML no están. Uno de ellos es el atributo target.

_blank

Es un enlace que cuando haces clic en él, la página de destino del enlace se abre en una pestaña nueva.

target="_blank"

_self

Es una etiqueta opcional, ya que de no usarse el browser asume como _self (sí mismo, la misma) la página donde va a ser abierto el link.

target="_self"

_parent

Abrirá el siguiente nivel arriba de un marco si se anidaron uno dentro del otro

target="_parent"

_top

El atributo de destino especifica dónde abrir el documento vinculado.

target="_top"
